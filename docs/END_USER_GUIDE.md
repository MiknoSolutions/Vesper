Vesper Quick Start

1. Download release zip and extract it.
2. Double-click Vesper.exe.
3. App starts in system tray near the clock.
4. Double-click tray icon to open window.
5. Open Settings and choose backend:
   - Local: download model first.
   - OpenAI API: paste your own API key.
   - Google Cloud: import your own service account JSON.
6. Select microphone, language, and hotkey.
7. Click Save.
8. Click into any text field and use your hotkey to dictate.

Tips
- Keep app running in tray.
- If transcription does not work, recheck backend setup and microphone.
- For Google Cloud, each user must provide their own credentials.
